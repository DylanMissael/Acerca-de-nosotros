Proyecto integrador

Swaply

Una plataforma pensada para que las personas coleccionistas puedan descubrir, intercambiar y conectar alrededor de sus objetos favoritos.

Conocer el proyecto Ver identidad visual

Definición del proyecto

Colecciona. Intercambia. Conecta.

Swaply organiza el intercambio de artículos coleccionables en una experiencia clara, confiable y social. El proyecto busca reducir la fricción para encontrar personas con intereses similares y facilitar acuerdos de intercambio.

Colecciona

Espacio para mostrar piezas, intereses y artículos disponibles.

Intercambia

Flujo enfocado en acuerdos claros entre usuarios.

Conecta

Comunidad para descubrir personas con gustos compatibles.

Identidad visual

Una marca dinámica, cercana y confiable.

La identidad toma como base el movimiento de intercambio del logotipo, con colores de alta energía para acciones principales y azul marino para dar orden y legibilidad.
